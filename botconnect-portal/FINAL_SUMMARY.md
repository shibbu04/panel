## ⚡ **Quick Customization**

### **Change Site Name:**
Edit `src/data/data.js`:
```javascript
export const siteData = {
  siteName: "Your Name Here",
  // ...
}
```

### **Change Colors:**
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: '#YOUR_COLOR',
  secondary: '#YOUR_COLOR',
  accent: '#YOUR_COLOR',
}
```

### **Update Pricing:**
Edit `src/data/data.js` → `pricingPlans` array
